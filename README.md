\# Dice Roller App 🎲



A simple Android dice roller app built with Kotlin and Jetpack Compose.



\## Features

\- Roll a dice by tapping a button

\- Displays dice faces 1 through 6

\- Clean Material Design UI with top app bar



\## Concepts Learned

\- Jetpack Compose UI

\- State management with mutableStateOf

\- Recomposition

\- Scaffold and TopAppBar

\- Column layout and Modifiers



\## Built With

\- Kotlin

\- Jetpack Compose

\- Material 3

\- Android Studio
